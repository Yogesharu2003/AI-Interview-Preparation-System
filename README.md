# 📌 Project Overview

The **AI Interview Preparation System** is a full-stack web application that helps job seekers improve their interview skills through AI-powered resume analysis, ATS scoring, mock interviews, coding practice, job recommendations, and performance tracking.

This project was developed using **Python, Flask, MySQL, HTML, CSS, JavaScript, and AI-based modules**.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/yourusername/AI_Interview_Preparation_System.git
```

Go to Project

```bash
cd AI_Interview_Preparation_System
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Linux

```bash
source venv/bin/activate
```

Install Packages

```bash
pip install -r requirements.txt
```

Run Application

```bash
python app.py
```
